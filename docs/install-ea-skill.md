# Install the ScientiaMesh EA Skill

The Executive Assistant skill lives at:

```text
skills/scientiamesh-ea
```

## From GitHub

For agents that support installing skills from a GitHub repository, use:

```text
github:ScientiaMesh/agent-kit/skills/scientiamesh-ea
```

For manual install:

```bash
git clone https://github.com/ScientiaMesh/agent-kit.git
cp -R agent-kit/skills/scientiamesh-ea "$CODEX_HOME/skills/"
```

## From Bundle

```bash
curl -fsSLO https://download.scientiamesh.app/skills/scientiamesh-ea/latest.tar.gz
tar -xzf latest.tar.gz
```

The extracted folder is:

```text
scientiamesh-ea/
```
